NONGKIS

Halaman Utama (index)
-Menampilkan Tempat yang tersedia
-Tidak bisa booking

Terdapat 3 pengguna:
-User
-Admin
-Owner



1. User(index/login, dashboard/login, notifikasi/login, booking, rute):
-Menampilkan Tempat dan data tempat yang sesuai kategori dari owner.
-Mengisi form booking tempat yang dipilih sesuai kapasitas kategori.
-Data form masuk ke Admin.
-Melihat notifikasi booking.
-Admin menerima data booking.
-Admin menghubungi owner.
-Owner memverifikasi tempat ke admin.
-Admin memberitahu ke user lewat notifikasi.
-Jika diterima booking, user di arahkan ke halaman rute dan melacak lokasi user kemudian mengarahkan user ke rute terdekat dari alamat tempat booking.
-Terdapat tombol konfirmasi dan pembatalan pada notifikasi booking dari user.
-Jika sudah melewati setengah jam dari waktu booking, maka bookingan langsung di batalkan.

2. Admin (index/admin, dashboard/admin, verifikasiOwner/admin, verifikasiBooking/admin).
-Akses melalui endpoint tertentu.
-Verifikasi admin.
-Menampilkan data tempat yang di tambahkan oleh owner.
-Menampilkan statistik data user(jumlah akun user dan owner(opsional), Jumlah tempat yang tambahkan, jumlah booking (diterima, belum dikonfirmasi).
-Memverifikasi data tempat yang ingin ditambahkan oleh owner.
-Memverifikasi data booking yang dibuat oleh user.
